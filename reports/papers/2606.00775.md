# GIRL-DETR: Gradient-Isolated Reinforcement Learning for Video Moment Retrieval

## 摘要

**论文元信息。** GIRL-DETR 的完整标题为 *GIRL-DETR: Gradient-Isolated Reinforcement Learning for Video Moment Retrieval*，作者为 Shihang Zhang、Mingjin Kuai、Ye Wei、Zhen Zhang、Wei Ji，arXiv ID 为 2606.00775，发布时间为 2026-06-02，类别为 cs.CV。论文链接为 <https://arxiv.org/abs/2606.00775>，PDF 链接为 <https://arxiv.org/pdf/2606.00775>。论文首页明确给出代码地址 `https://github.com/Z-Shihang/GIRL-DETR`，并说明 “The code is available”（见 PAGE 1）。仓库可公开访问，README 标注其为官方 PyTorch 实现，同时说明源码、预训练模型和数据准备脚本正在逐步发布；因此本文的代码分析仅基于当前仓库中可确认的已公开文件。

**一句话总结。** GIRL-DETR 将视频时刻检索（Video Moment Retrieval, VMR）拆成“稳定表征学习”和“指标对齐优化”两个阶段：先用监督学习获得跨模态表征，再冻结骨干网络，仅用三 阶段渐进式强化学习更新检测头，直接优化非可微的 temporal IoU（tIoU），从而缓解代理损失与评价指标不一致导致的边界定位退化（见 PAGE 1、PAGE 2、PAGE 5、PAGE 6）。

本文的核心贡献可以概括为三点。第一，论文把轻量级 VMR 框架中的代理损失退化（loss degradation）明确建模为优化目标与评价指标之间的错配：监督阶段常用的 $L_1$、GIoU、分类损失给出的是连续可微梯度，但最终评价依赖离散排序和非可微 tIoU（见 PAGE 1、PAGE 5）。第二，模型在架构上引入 Cross-Modal Interaction（CMI，跨模态交互）和 Text-Guided Gating（TGG，文本引导门控），使视频片段和查询文本在进入解码器前建立更强的语义对齐（见 PAGE 3、PAGE 4）。第三，论文提出 Gradient Isolation（梯度隔离）与 Three-stage Progressive Reinforcement Learning（TPRL，三阶段渐进式强化学习），冻结跨模态骨干，只让检测头承受高方差 RL 梯度，从而在不破坏表征流形的前提下直接优化 tIoU（见 PAGE 4、PAGE 5、PAGE 6）。
