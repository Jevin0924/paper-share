C-MambaPose 试图解决 WiFi-based 3D Human Pose Estimation（基于 WiFi 的三维人体姿态估计）在跨环境部署时的泛化问题。它的核心思路不是把 WiFi CSI 当作普通实值张量处理，而是保留 CSI phase（相位）的物理复数结构，经 phase sanitization（相位校正）、complex-valued patchification（复值切片）、Complex Mamba sequence modeling（复值 Mamba 序列建模）和 GraFormer GCN decoder（图卷积骨架解码器），将非结构化 WiFi 序列映射为 17 个三维人体关节点，见 PAGE 1、PAGE 2、PAGE 5。

一句话总结：本文提出 C-MambaPose，通过“相位保真复值表征 + 线性复杂度 Mamba 序列建模 + 骨架拓扑约束”提升 WiFi 姿态估计在跨房间环境中的泛化能力，并在 MM-Fi Setting 3 上达到 298.5 mm MPJPE、3.78M 参数规模，见 PAGE 6、PAGE 7。

## 背景与动机

三维人体姿态估计（3D Human Pose Estimation, 3D HPE）的传统主流路线依赖 RGB、深度或多视角视觉输入。视觉方案在公开基准上通常精度较高，但在真实部署中面临三个结构性问题：弱光条件下图像质量下降、遮挡导致关键点不可见、摄像头采集带来隐私风险。论文在引言中明确把 smart home automation、healthcare monitoring 和 human-computer interaction 作为应用背景，并指出 camera-based approaches 受 poor lighting、occlusions 和 privacy concerns 限制，见 PAGE 1。

WiFi sensing（WiFi 感知）提供了一条非视觉路线。Channel State Information（CSI，信道状态信息）记录无线信号在传播路径中的多径反射变化，人体运动会改变反射路径，从而在幅度和相位中留下时空动态。与摄像头不同，WiFi CSI 不直接采集可见图像，因此更适合隐私敏感、遮挡严重或光照不可控的场景，见 PAGE 1。

然而，WiFi HPE 的难点也正来自无线传播本身。房间布局、家具、墙面材质和多径散射会显著改变 CSI 分布，使模型在一个房间训练、另一个房间测试时出现 severe domain shift（严重域偏移）。论文将 cross-environment split（跨环境划分）定义为最具挑战性的设置：使用三个环境训练、一个环境测试，见 PAGE 6。

既有方法的第一个限制是 phase information（相位信息）处理不足。论文指出，一些方法仅使用 amplitude（幅度），或者把复数 CSI 的 real part（实部）和 imaginary part（虚部）当作独立实值通道处理；而 raw phase（原始相位）又会受到 Sampling Frequency Offset（SFO，采样频偏）、Carrier Frequency Offset（CFO，载波频偏）和 Phase-Locked Loop（PLL，相位锁定环）噪声污染，见 PAGE 1、PAGE 2。直接丢弃相位会丢失与人体微小位移和 Doppler shifts（多普勒频移）相关的物理信息。

第二个限制是模型复杂度与泛化之间的矛盾。论文指出 MetaFi++ 使用较大的 CNN/Transformer backbone，参数量达到 26.52M，容易对背景多径噪声过拟合；DT-Pose 将参数量降至 3.20M，但缺少显式骨架物理正则，可能在未见环境中产生解剖学不合理的姿态，见 PAGE 1、PAGE 6。

因此，C-MambaPose 的出发点可以概括为三个问题：如何在去除相位误差的同时保留相位几何结构；如何用较小模型建模长程时序依赖；如何把 WiFi token 转换为符合人体骨架拓扑的 3D joints。论文的方案分别对应 CSI phase sanitization、Spatiotemporal Complex Mamba encoder 和 Regularized GraFormer decoder，见 PAGE 2、PAGE 5。

## 预备知识

CSI（Channel State Information）通常表示无线信道在不同天线、子载波和时间帧上的复数响应。论文将原始复数 CSI 写为 $H \in \mathbb{C}^{A \times S \times T}$，其中 $A$ 是接收天线数，$S$ 是子载波数，$T$ 是帧序列长度，见 PAGE 2。对于 MM-Fi 设置，论文使用 1 个发射天线、3 个接收天线、114 个子载波，并将连续帧聚合为 $3 \times 114 \times 10$ 的输入样本，见 PAGE 5。

复值表示的关键在于：幅度 $|H|$ 描述信号强弱，相位 $\Theta$ 描述信号角度。人体运动引起的路径长度变化会体现在相位变化中，因此相位不是可随意丢弃的附属信息。但 raw CSI phase 受到硬件偏移污染，需要先做 unwrap（相位展开）和线性误差去除，见 PAGE 2。

Mamba 属于 selective state-space model（选择性状态空间模型）。与 Transformer 的全局注意力不同，SSM 通过状态递推建模序列，通常具有线性时间复杂度。论文将 Mamba 扩展到 complex-valued sequence（复值序列），并让状态参数由信号 magnitude（幅值包络）驱动，见 PAGE 4。

GraFormer/GNN 部分提供 human skeleton graph（人体骨架图）先验。人体关节不是独立点，关节之间有固定骨骼连接关系。论文将关节定义为图 $G=(V,E)$，其中 $V$ 是 joints，$E$ 是 bones，并通过 Chebyshev Graph Convolution 和 bone length loss 约束输出姿态的解剖一致性，见 PAGE 5。

## 方法详解

### 1. CSI 相位清洗与解耦表征

C-MambaPose 的第一步是处理 raw CSI phase。论文将原始相位 $\Theta_{\mathrm{raw}} \in \mathbb{R}^{A \times S \times T}$ 视为被线性误差污染的信号，并沿子载波维度进行相位展开：

$$
\theta^{\mathrm{unwrapped}}_{i,s,t} = \mathrm{unwrap}(\theta^{\mathrm{raw}}_{i,s,t})
$$

这里 $i$ 表示天线索引，$s$ 表示子载波索引，$t$ 表示时间步；该公式的含义是先消除 $2\pi$ 周期跳变，使相位序列在子载波轴上连续，见 PAGE 2。

随后，论文将展开后的相位建模为线性误差项与 clean phase（干净相位）的叠加：

$$
\theta^{\mathrm{unwrapped}}_{i,s,t}=s \cdot m_{i,t}+c_{i,t}+\theta^{\mathrm{clean}}_{i,s,t}
$$

其中 $m_{i,t}$ 是由 SFO/CFO 引入的 phase slope（相位斜率），$c_{i,t}$ 是 constant phase offset（常数相位偏移）。通俗地说，论文假设硬件误差主要表现为沿子载波方向的一条直线，而人体运动相关的相位变化保留在线性残差中，见 PAGE 2。

线性斜率和偏移通过 least-squares linear regression（最小二乘线性回归）估计。论文给出 $m_{i,t}$ 和 $c_{i,t}$ 的闭式形式，见 PAGE 2：

$$
m_{i,t} =
\frac{
N\sum_{s=0}^{N-1}s\theta^{\mathrm{unwrapped}}_{i,s,t}
-\left(\sum_{s=0}^{N-1}s\right)\left(\sum_{s=0}^{N-1}\theta^{\mathrm{unwrapped}}_{i,s,t}\right)
}{
N\sum_{s=0}^{N-1}s^2-\left(\sum_{s=0}^{N-1}s\right)^2
}
$$

$$
c_{i,t} =
\frac{\sum_{s=0}^{N-1}\theta^{\mathrm{unwrapped}}_{i,s,t}-m_{i,t}\sum_{s=0}^{N-1}s}{N}
$$

这两个公式的直观含义是：在每个天线和时间步上，用所有子载波拟合一条误差直线，再把这条直线从原相位中减掉。

clean phase 的计算为：

$$
\theta^{\mathrm{clean}}_{i,s,t}
=
\theta^{\mathrm{unwrapped}}_{i,s,t}
-
(s \cdot m_{i,t}+c_{i,t})
$$

该式说明 C-MambaPose 并不是直接学习 raw phase，而是学习去除硬件线性偏移后的相位残差，见 PAGE 2。

相位清洗后，论文没有把 phase 和 amplitude 立即融合，而是构建 decoupled representation（解耦表征）。对相位分支，clean phase 被映射到 unit circle（单位圆）：

$$
P=\cos(\Theta^{\mathrm{clean}})+j\sin(\Theta^{\mathrm{clean}})\in\mathbb{C}^{A\times S\times T}
$$

该公式的意义是：相位不再作为一个普通实数角度输入，而是作为复平面单位圆上的点输入，从而保留角度关系和相位连续性，见 PAGE 2。

为了适配标准深度学习张量，论文将实部和虚部沿天线通道拼接：

$$
X_{\mathrm{phase}}=[\cos(\Theta^{\mathrm{clean}});\sin(\Theta^{\mathrm{clean}})]
$$

若接收天线数 $A=3$，则 phase input channel 为 $C^p_{\mathrm{in}}=6$，分别对应 3 个天线的 cos 和 sin 分量，见 PAGE 2。

对 amplitude 分支，论文采用 sample-wise standardization：

$$
X_{\mathrm{amp}}=\frac{A-\mu_A}{\sigma_A+\eta}
$$

其中 $A=|H|$ 是 CSI 幅度，$\mu_A$ 和 $\sigma_A$ 分别是在子载波和时间维度上的均值与标准差，$\eta$ 是稳定项。该式的目标是降低静态环境差异，使幅度输入更关注相对变化而非房间特定偏置，见 PAGE 2。

### 2. Parallel complex feature extraction 与 Complex-DSKConv

用途：下图用于说明 C-MambaPose 的整体数据流，从 raw CSI 到 phase/amplitude 解耦、Complex Mamba 编码、joint-query mapping 和 GraFormer 解码。

读图要点：重点观察模型并非单一路径，而是 phase branch 与 amplitude branch 并行处理，随后在 complex space late fusion，再进入 Complex Mamba 和 GCN decoder。

![Fig. 1 framework overview](../assets/2606.13700/figure-01-p3.png)

支撑的判断：Fig. 1 支撑本文对三阶段架构的解释，即 CSI phase sanitization and decoupled formulation、Spatiotemporal Complex Late-Fusion Mamba Encoder、Cross-Attention Joint-Query Mapping + Regularized GraFormer Decoder，见 PAGE 2、PAGE 3。

在 phase encoder branch 中，论文使用 complex-valued 2D convolution 将 $X_{\mathrm{phase}}$ 投影到复值隐藏空间：

$$
Z_{\mathrm{phase}}=W^p_{\mathrm{conv}}\circledast X_{\mathrm{phase}}+b^p_{\mathrm{conv}}
$$

这里 $\circledast$ 表示 complex convolution。该公式说明，模型在卷积层就保留复数运算结构，而不是把相位拆成互不相关的实值特征，见 PAGE 3。

随后使用 Complex-ModReLU 激活：

$$
U_{\mathrm{phase}}
=
\max(0,|Z_{\mathrm{phase}}|+b^p_{\mathrm{act}})
\odot
\frac{Z_{\mathrm{phase}}}{|Z_{\mathrm{phase}}|+\eta}
$$

这个公式的含义是：非线性门控作用在 magnitude 上，但输出方向仍沿原复数相位方向，因此比普通 ReLU 更适合保留 phase coherence（相位一致性），见 PAGE 3。

Complex-DSKConv 是本文的重要局部建模模块。它将 Selective Kernel Networks 思路扩展到复值域，用多个 dilation branch（不同膨胀率分支）捕捉不同速度的人体运动。论文先将多个分支输出求和：

$$
U_{\mathrm{sum}}=\sum_{b=1}^{B}U_b
$$

再计算复数幅值包络：

$$
|U_{\mathrm{sum}}|=\sqrt{(U^R_{\mathrm{sum}})^2+(U^I_{\mathrm{sum}})^2+\eta}
$$

这一步把复值分支输出转换为可用于注意力选择的幅值描述，见 PAGE 3。

每个分支的 channel-wise attention weight 为：

$$
\alpha_{b,c}
=
\frac{\exp(W_{2,b,c}z)}
{\sum_{k=1}^{B}\exp(W_{2,k,c}z)}
$$

最终的复值选择输出为：

$$
U_{\mathrm{select}}
=
\sum_{b=1}^{B}\alpha_b\odot U_b
$$

该机制的意义是：当动作速度、人体反射路径变化尺度不同，模型可以动态选择更合适的 receptive field（感受野），而不是固定卷积尺度，见 PAGE 3。

Amplitude branch 则先进行 mean-time subtraction：

$$
X'_{\mathrm{amp}}=X_{\mathrm{amp}}-\mathrm{mean}_t(X_{\mathrm{amp}})
$$

该式意在移除时间方向上的静态环境偏置，再经过实值 2D convolution、ReLU、real-valued DSKConv 和 patchifying convolution 得到 amplitude patch tokens，见 PAGE 3、PAGE 4。

### 3. Late fusion in complex space

C-MambaPose 的融合策略是 late fusion，而非 early fusion。论文将 amplitude tokens $F_{\mathrm{amp}}$ 投影为虚部为零的复数：

$$
F'_{\mathrm{amp}}=F_{\mathrm{amp}}+j0
$$

这表示幅度信息作为复平面的实轴分量加入模型，而不会伪造不存在的相位，见 PAGE 4。

若 phase stream 为：

$$
F_{\mathrm{phase}}=F^R_{\mathrm{phase}}+jF^I_{\mathrm{phase}}
$$

则融合后的实部和虚部分别为：

$$
F^R_{\mathrm{fused}}=[F^R_{\mathrm{phase}};F_{\mathrm{amp}}],
\quad
F^I_{\mathrm{fused}}=[F^I_{\mathrm{phase}};0]
$$

该式说明：phase 分支贡献真实复数结构，amplitude 分支只补充实值强度描述。这样既避免早期融合破坏相位关系，又让幅度包络参与后续序列建模，见 PAGE 4。

融合后的 tokens 被展平为长度 $N_p=S_p\times T_p$ 的序列，并通过 complex-valued linear layer 投影到 encoder embedding dimension $D$：

$$
Z_{\mathrm{init}}=\mathrm{ComplexLinear}(F_{\mathrm{fused}})\in\mathbb{C}^{N_p\times D}
$$

随后加入 complex positional embeddings 和 class token：

$$
Z_0=[Z_{\mathrm{init}};t_{\mathrm{cls}}]+E_{\mathrm{pos}}
$$

其中 $t_{\mathrm{cls}}$ 用于聚合全局序列上下文，但在后续 joint-query mapping 中被排除，以便关节查询关注局部 joint-specific spatiotemporal features，见 PAGE 4、PAGE 5。

### 4. Spatiotemporal Complex Mamba encoder

用途：下图用于强调 Fig. 1 中 Complex Mamba block 的内部操作，包括 magnitude extraction、parameter projection、RoPE、split-complex ZOH selective scan 和 magnitude projection。

读图要点：关注 Complex Mamba 并非直接对实值幅度做 SSM，而是在复值序列内部执行选择性状态更新，最后才投影到实值 magnitude space。

![Fig. 1 complex mamba details](../assets/2606.13700/figure-02-p3.png)

支撑的判断：该图支撑“相位关系在 Mamba 编码阶段内部被保留”的判断；论文同时通过 Phase Bottleneck 消融证明，如果在 Mamba 前就投影为实值 magnitude，MPJPE 会从 298.5 mm 退化到 335.0 mm，见 PAGE 3、PAGE 7。

Complex Mamba 的输入是复值状态 $x_t=x^R_t+jx^I_t$。论文首先从 magnitude 估计选择性 SSM 参数：

$$
|x_t|=\sqrt{(x^R_t)^2+(x^I_t)^2+\eta}
$$

$$
[\Delta_t,B_t,C_t]=\mathrm{Split}(W_{\mathrm{proj}}|x_t|)
$$

其中 $\Delta_t$ 是 step size（步长），$B_t$ 和 $C_t$ 是状态投影向量。该设计的含义是：状态空间模型的选择性参数由物理信号强度驱动，使模型根据人体反射强弱动态调整序列感受野，见 PAGE 4。

论文还将 RoPE（Rotary Position Embedding，旋转位置编码）施加到状态参数 $B_t$ 和 $C_t$ 上。对 $D_{\mathrm{state}}$ 维实向量 $v_t$，相邻元素成对旋转：

$$
v^{\mathrm{rot}}_{t,2k}
=
v_{t,2k}\cos(t\theta_k)-v_{t,2k+1}\sin(t\theta_k)
$$

$$
v^{\mathrm{rot}}_{t,2k+1}
=
v_{t,2k}\sin(t\theta_k)+v_{t,2k+1}\cos(t\theta_k)
$$

该公式的作用是将相对时空位置先验注入状态参数，使 SSM 不只依赖内容，也感知 token 的序列位置，见 PAGE 4。

在 split-complex selective scan 中，论文采用 Zero-Order Hold（ZOH，零阶保持）离散化。对通道 $d$ 和状态维 $n$：

$$
\bar{A}_{t,d,n}=\exp(\Delta_{t,d}A_{d,n})
$$

$$
\bar{B}_{t,d,n}
=
(\Delta_{t,d}A_{d,n})^{-1}
(\exp(\Delta_{t,d}A_{d,n})-1)
\Delta_{t,d}B^{\mathrm{rot}}_{t,d,n}
$$

这两个公式的含义是将连续状态空间参数转换为离散递推参数，用于逐时间步执行状态更新，见 PAGE 4。

由于 $\bar{A}$、$\bar{B}$ 和 $C^{\mathrm{rot}}$ 为实值，复数状态递推被拆成实部和虚部两条并行路径：

$$
h^R_{t,d,n}
=
\bar{A}_{t,d,n}h^R_{t-1,d,n}
+
\bar{B}_{t,d,n}x^R_{t,d}
$$

$$
h^I_{t,d,n}
=
\bar{A}_{t,d,n}h^I_{t-1,d,n}
+
\bar{B}_{t,d,n}x^I_{t,d}
$$

这说明实部和虚部共享相同状态动力学参数，只在输入轨迹上分离，因此能够维持复数相位结构的一致性，见 PAGE 4。

输出也分别计算：

$$
y^R_{t,d}
=
\sum_{n=1}^{D_{\mathrm{state}}}C^{\mathrm{rot}}_{t,d,n}h^R_{t,d,n}
+
D_fx^R_{t,d}
$$

$$
y^I_{t,d}
=
\sum_{n=1}^{D_{\mathrm{state}}}C^{\mathrm{rot}}_{t,d,n}h^I_{t,d,n}
+
D_fx^I_{t,d}
$$

论文进一步将实部和虚部沿 batch dimension stack，以便在单个 GPU selective scan kernel 中并行执行两条轨迹，见 PAGE 4。

为降低跨环境过拟合，论文提出 Shared-Mask Complex Dropout：

$$
\mathrm{ComplexDropout}(x)
=
[x^R\odot M_{\mathrm{drop}};x^I\odot M_{\mathrm{drop}}]\cdot\frac{1}{1-p}
$$

该式的关键是实部和虚部共享同一个 dropout mask。若实部和虚部分别随机丢弃，会改变 $\angle x=\arctan(x^I/x^R)$，造成随机相位扰动；共享 mask 则保持相位角关系，见 PAGE 5。

最后，Complex Mamba 的输出被投影为 magnitude representation：

$$
Z_{\mathrm{out}}=|Z_L|=\sqrt{(Z^R_L)^2+(Z^I_L)^2+\eta}
$$

该步骤表示复值建模发生在 encoder 内部，而 decoder 接收实值幅值特征，见 PAGE 5。

### 5. Cross-attention joint-query mapping

Mamba encoder 输出的是 sequence tokens，不天然对应人体关节。C-MambaPose 因此定义 $J$ 个 learnable joint-query embeddings（可学习关节查询）$Q\in\mathbb{R}^{J\times D}$，用 Multi-Head Cross-Attention（MHCA，多头交叉注意力）从 token 序列中为每个关节聚合相关特征，见 PAGE 5。

论文给出 joint mapping 公式：

$$
Z_{\mathrm{joint}}
=
\mathrm{LN}
\left(
Q+
\mathrm{softmax}
\left(
\frac{(QW_Q)(MW_K)^T}{\sqrt{d_k}}
\right)
\cdot MW_V
\right)
$$

其中 $M=Z_{\mathrm{out},1:N_p}$，即排除 class token 后的局部时空序列；$W_Q,W_K,W_V$ 为投影矩阵；$d_k$ 为 key dimension；LN 为 Layer Normalization。该式表示每个 joint query 可以在所有 spatiotemporal tokens 中选择与自身最相关的子载波-时间特征，见 PAGE 5。

这一设计与直接 pooling 输出坐标不同。直接 pooling 假设全局特征可以一次性回归所有关节，而 joint-query mapping 为每个关节提供独立查询入口，更适合从无显式空间网格结构的 WiFi token 中恢复人体骨架。

### 6. Regularized GraFormer decoder 与训练损失

joint features 被送入 Regularized GraFormer Decoder。论文将人体定义为图 $G=(V,E)$，其中 vertices $V$ 是 $J$ 个关节，edges $E$ 是骨骼连接，见 PAGE 5。Chebyshev Graph Convolution 的更新公式为：

$$
Z^{(l+1)}
=
\sum_{k=0}^{K_{\mathrm{cheb}}-1}
T_k(\tilde{L})Z^{(l)}W^{\mathrm{GCN}}_k
$$

其中 $\tilde{L}=2L/\lambda_{\max}-I$ 是 scaled normalized Laplacian，$T_k$ 是 $k$ 阶 Chebyshev polynomial。该公式说明，关节特征更新不只来自自身，也来自骨架图上的局部邻域，见 PAGE 5。

训练目标由 absolute coordinate loss 和 kinematic bone loss 组成：

$$
\mathcal{L}
=
\mathcal{L}_{\mathrm{abs}}
+
\lambda_{\mathrm{bone}}\mathcal{L}_{\mathrm{bone}}
$$

绝对位置损失为：

$$
\mathcal{L}_{\mathrm{abs}}
=
\frac{1}{J}
\sum_{j=1}^{J}
\|\hat{Y}_j-Y_j\|_2^2
$$

该项直接惩罚预测关节坐标 $\hat{Y}_j$ 与真实坐标 $Y_j$ 的距离误差，见 PAGE 5。

骨长一致性损失为：

$$
\mathcal{L}_{\mathrm{bone}}
=
\frac{1}{|E|}
\sum_{(u,v)\in E}
\left(
\|\hat{Y}_u-\hat{Y}_v\|_2
-
\|Y_u-Y_v\|_2
\right)^2
$$

该项不直接约束某个关节的位置，而是约束相邻关节之间的骨长差异，目的是防止跨环境噪声导致 unnatural bone stretching（不自然骨骼拉伸），见 PAGE 5。

### 7. 方法组件与证据完整性

C-MambaPose 的创新可以归纳为三点。第一，phase-preserving complex representation：通过相位展开、线性误差消除、单位圆复数映射和 Complex-ModReLU 保留相位物理结构，见 PAGE 2、PAGE 3。第二，complex-valued Mamba sequence modeling：在复值空间内进行 selective scan，并用 signal magnitude 产生状态参数，见 PAGE 4、PAGE 5。第三，joint-query + GraFormer skeleton prior：用交叉注意力把 sequence tokens 映射为 joints，再用 GCN 和 bone loss 强化骨架约束，见 PAGE 5。

证据不足处也必须明确：给定 figures 只包含 Fig. 1 的两个切片，没有提供独立 Fig. 2 或 Fig. 3 图像；论文正文中可引用 Table I、Table II、Table III、Table IV，但图片文件只能展示给定的两个 Fig. 1 相关路径。代码方面，论文声明公开代码，但给定材料未包含 README、源码文件或配置文件，因此不能给出“论文方法 ↔ 源码函数”的逐行对应分析。

## 实验分析

### 1. 实验设置

论文在 MM-Fi dataset 上评估。MM-Fi 被描述为 largest public multi-modal dataset for WiFi-based human sensing，包含 40 个 subjects、4 个 environments 和 14 个 daily activities；Protocol 1 关注 14 类日常动作，Protocol 2 关注 13 类康复动作，见 PAGE 5。输入 CSI 工作在 5 GHz、40 MHz 带宽、1 发射天线和 3 接收天线，CSI 采集 114 个子载波，样本大小为 $3\times114\times10$，姿态标注为 17 个 3D joints，见 PAGE 5。

论文采用三种划分。Setting 1 为 random split，训练测试比例 3:1；Setting 2 为 cross-subject split，32 个 subject 训练、8 个 subject 测试；Setting 3 为 cross-environment split，三个环境训练、一个环境测试，见 PAGE 6。评价指标包括 MPJPE、PA-MPJPE、PCK@20 和 PCK@50，见 PAGE 6。

实现细节包括：PyTorch、NVIDIA RTX 4070、AdamW、初始学习率 $3\times10^{-4}$、weight decay 0.02、cosine annealing 50 epochs、batch size 256、$\lambda_{\mathrm{bone}}=0.1$。模型参数包括 $L=2$ 个 complex SSM blocks、$d_{\mathrm{state}}=16$、$d_{\mathrm{conv}}=4$、expansion factor $E=2$、complex dropout $p=0.3$、joint-query dimension 256、4 attention heads、GraFormer 4 graph layers、hidden dimension 128、GCN dropout 0.4、Complex-DSKConv dilations [1,2]，见 PAGE 6。

### 2. 主要结果：跨环境 Setting 3 是最关键证据

| Setting 3 / Protocol 1 | PCK@20 ↑ | PCK@50 ↑ | MPJPE ↓ | PA-MPJPE ↓ |
|---|---:|---:|---:|---:|
| MetaFi++ | 9.3 | 55.1 | 367.8 | 121.0 |
| HPE-Li | 4.3 | 47.8 | 381.1 | 110.3 |
| GraphPose-Fi | 12.9 | 67.2 | 302.7 | 103.0 |
| DT-Pose | 10.7 | 58.8 | 332.7 | 105.1 |
| C-MambaPose | 15.3 | 67.1 | 298.5 | 102.2 |

表格解读：Setting 3 是跨环境测试，最能体现泛化能力。C-MambaPose 在 Protocol 1 上取得 298.5 mm MPJPE，优于 GraphPose-Fi 的 302.7 mm 和 DT-Pose 的 332.7 mm；PCK@20 也达到 15.3，为该表最高值。PCK@50 略低于 GraphPose-Fi 的 67.2，仅差 0.1，但 MPJPE 和 PA-MPJPE 更优，说明其整体定位误差和结构误差更稳。数据来自 Table I，见 PAGE 7；论文正文也在 PAGE 6 对 Setting 3 的 298.5 mm、67.1% PCK@50 和 15.3% PCK@20 做了解读。

| Setting 3 / Protocol 2 | PCK@20 ↑ | PCK@50 ↑ | MPJPE ↓ | PA-MPJPE ↓ |
|---|---:|---:|---:|---:|
| MetaFi++ | 5.3 | 45.9 | 360.2 | 117.2 |
| HPE-Li | 4.2 | 40.3 | 378.2 | 104.0 |
| GraphPose-Fi | 5.3 | 42.0 | 373.4 | 105.4 |
| DT-Pose | 4.4 | 49.7 | 338.3 | 102.0 |
| C-MambaPose | 8.4 | 56.9 | 317.6 | 101.1 |

表格解读：Protocol 2 面向 rehabilitation exercises，C-MambaPose 在 Setting 3 上同样领先：MPJPE 为 317.6 mm，低于 DT-Pose 的 338.3 mm；PCK@50 为 56.9，高于 DT-Pose 的 49.7；PA-MPJPE 为 101.1，也优于其余方法。该结果说明优势不是只在日常动作协议中成立，而是在另一类动作集合上仍然保持，见 Table I，PAGE 7。

论文还观察到一个重要现象：从 Setting 1 到 Setting 3，MPJPE 从 165.1 mm 上升到 298.5 mm，但 PA-MPJPE 只从 100.8 mm 变化到 102.2 mm，见 PAGE 6、PAGE 7。这意味着跨环境域偏移主要影响 absolute localization（绝对位置），但 skeletal structure（相对骨架结构）被较好保持。该现象支持 GraFormer decoder 与 bone loss 的设计动机。

### 3. 模型复杂度与速度

| Model | Params (M) ↓ | Latency (ms) ↓ | MPJPE ↓ | PCK@50 ↑ |
|---|---:|---:|---:|---:|
| MetaFi++ | 26.52 | 59.94 | 367.8 | 55.1 |
| GraphPose-Fi | 22.43 | 62.61 | 302.7 | 67.2 |
| DT-Pose | 3.20 | 4.23 | 332.7 | 58.8 |
| HPE-Li | 0.86 | 2.24 | 381.1 | 47.8 |
| C-MambaPose | 3.78 | 25.43 | 298.5 | 67.1 |

表格解读：C-MambaPose 的参数量为 3.78M，相比 GraphPose-Fi 的 22.43M 减少 83.1%，相比 MetaFi++ 的 26.52M 减少 85.7%；同时在 Setting 3 上 MPJPE 为 298.5 mm，是该表最低。延迟方面，25.43 ms 明显慢于 DT-Pose 和 HPE-Li，但快于 MetaFi++ 与 GraphPose-Fi，并且低于 30 ms 实时阈值。论文将额外延迟归因于 sequential selective-scan recurrence 和 split-complex batch doubling，见 Table II，PAGE 6。

这一结果说明，C-MambaPose 的主要价值不是极致低延迟，而是在较小模型规模下换取跨环境准确性。对业务应用而言，它更适合“隐私/遮挡/弱光条件下需要较可靠姿态”的场景，而不是对毫秒级延迟极敏感的轻量设备场景。该判断基于 PAGE 1 的应用动机和 PAGE 6 的复杂度数据。

| Configuration | MPJPE ↓ | PA-MPJPE ↓ |
|---|---:|---:|
| Full Model | 298.5 | 102.2 |
| w/o Complex Representation | 365.5 | 103.5 |
| w/o Complex-DSKConv | 339.5 | 103.9 |
| w/o GraFormer (MLP decoder) | 358.9 | 103.2 |
| Phase Bottleneck (Mag before Mamba) | 335.0 | 102.3 |
| w/o RoPE | 312.4 | 102.9 |

表格解读：最大的退化来自去掉 complex representation，MPJPE 从 298.5 mm 退化到 365.5 mm，说明相位信息对跨环境泛化非常关键。去掉 GraFormer 后退化到 358.9 mm，说明骨架拓扑先验不是装饰性模块，而是显著影响 absolute localization。Phase Bottleneck 在 Mamba 前将复值特征提前转为实值 magnitude，使 MPJPE 退化到 335.0 mm，直接支持“必须在序列建模内部保留相位关系”的主张。该表来自 Table III，见 PAGE 7。

RoPE 和 Shared-Mask Complex Dropout 的贡献也有实验证据。去掉 RoPE 后 MPJPE 为 312.4 mm，说明状态转移中的相对位置编码有助于追踪 sequential body movements；去掉 shared mask 后 MPJPE 为 321.8 mm，说明独立 dropout 会破坏 real/imaginary phase angle，见 PAGE 7。

## 讨论

C-MambaPose 的适用边界首先由输入模态决定。它不是视觉姿态估计模型，不能直接复用现有 RGB keypoint detector 或 image backbone。它依赖 WiFi CSI 采集链路，包括 5 GHz、40 MHz、1Tx/3Rx、114 subcarriers、短时帧聚合等设置，见 PAGE 5。因此，它更适合作为视觉受限场景下的补充感知方案，而非通用替代所有视觉 HPE 系统。

从方法论上看，论文的核心价值在于把 WiFi 信号的物理结构注入深度模型。phase sanitization 处理硬件误差，unit-circle mapping 保留相位角，complex Mamba 在序列内部维持 real/imaginary coherence，GraFormer 则将输出限制在人类骨架拓扑上。这条链路比单纯扩大模型容量更符合 WiFi sensing 的信号特性，见 PAGE 2 至 PAGE 5。

从业务价值看，论文所强调的 privacy preservation、robustness against occlusion and poor lighting 对智能家居、医疗监护和人机交互场景有现实意义，见 PAGE 1。但部署风险同样明确：WiFi CSI 的硬件、标定、同步和环境适配成本与视觉系统差异很大；跨环境 Setting 3 虽有提升，但 MPJPE 仍接近 300 mm 量级，不能简单等价于视觉 3D pose 的精细动作捕捉。

代码层面，论文声明代码公开，见 PAGE 1；但给定材料没有 README、训练脚本、模型定义或配置文件。证据不足以确认 Complex-ModReLU、Complex-DSKConv、split-complex selective scan、GraFormer dropout 和 loss weighting 在源码中的具体实现。因此本文不提供代码段，也不声称已验证论文方法与源码逐行一致。

## 局限分析

作者自述的未来工作包含三点：扩展到 multi-person pose estimation，需要处理多人的 WiFi reflection paths 空间重叠；研究 transfer learning 和 domain adaptation，以进一步降低 completely unseen rooms 下的误差；部署到 edge computing devices 以验证实时性能和智能家居实用性，见 PAGE 8。这些实际上也构成当前方法的未解决问题。

第一项局限是 single-person assumption（单人假设）。论文实验设置来自 MM-Fi 的人体感知任务，但结论没有证明多人体场景下的反射路径分离能力。WiFi 多径信号本身是混合观测，多人同时运动会造成反射叠加，joint-query mapper 是否能区分不同个体没有证据支持，见 PAGE 8。

第二项局限是 cross-environment 仍未完全解决。Setting 3 的 MPJPE 虽然优于 baselines，但 Protocol 1 仍为 298.5 mm，Protocol 2 为 317.6 mm，远高于 Setting 1 的 165.1 mm 和 194.8 mm，见 PAGE 7。这说明 C-MambaPose 缓解了域偏移，但尚未消除域偏移。

第三项局限是 latency trade-off。C-MambaPose 的 25.43 ms 延迟低于 30 ms，但明显高于 DT-Pose 的 4.23 ms 和 HPE-Li 的 2.24 ms，见 PAGE 6。对低功耗 edge device 或高频连续姿态追踪场景，split-complex selective scan 带来的额外计算可能成为工程瓶颈；论文也只报告 RTX 4070 上的训练和推理设置，见 PAGE 6。

第四项局限是证据材料中缺少公开源码核验。虽然论文声称代码公开，见 PAGE 1，但本次分析无法读取仓库文件。对复值运算、SSM kernel、dropout mask 共享和 GraFormer 强 dropout 这类实现敏感模块，源码细节会影响可复现性。当前只能依据论文公式和表格评价方法，不能确认工程实现质量。

## 结论

C-MambaPose 是一篇围绕 WiFi CSI 物理属性设计的姿态估计论文。它把 phase sanitization、complex-valued representation、Complex Mamba、joint-query mapping 和 GraFormer skeleton regularization 组合成一条完整链路，目标是提升 cross-environment WiFi 3D HPE 的泛化能力。实验上，它在 MM-Fi Setting 3 上以 3.78M 参数达到 298.5 mm MPJPE，并在多项消融中证明 complex representation、Complex-DSKConv、GraFormer、RoPE 和 shared-mask dropout 都对性能有贡献，见 PAGE 6、PAGE 7。

从研究视角看，本文的贡献不在于提出单个通用模块，而在于把无线信号处理中的相位校正、复值神经网络、状态空间序列建模和人体骨架先验串联为面向跨环境泛化的系统设计。从应用视角看，它为隐私敏感、弱光、遮挡环境中的人体姿态感知提供了有价值的非视觉备选方案，但仍受限于 WiFi CSI 硬件链路、单人设置、跨环境残余误差和源码可核验性。
